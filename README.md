# 🎯 Real-Time Object Detection and Tracking

A browser-based AI-powered object detection and tracking system built using TensorFlow.js and the COCO-SSD model. This project performs real-time object detection through a webcam feed and displays bounding boxes, confidence scores, object counts, and detection history directly in the browser.

---

## 🚀 Features

* Real-time object detection using webcam
* TensorFlow.js and COCO-SSD integration
* Live object tracking visualization
* Bounding boxes around detected objects
* Confidence score display
* Detection history log
* Object counting system
* FPS (Frames Per Second) monitoring
* Screenshot capture functionality
* Adjustable confidence threshold
* Modern responsive UI

## ⚙️ Installation & Usage

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/ai-object-detection-tracker.git
```

### 2. Open the Project

Navigate to the project folder and launch the HTML file using a local server.

### 3. Allow Camera Access

Grant webcam permissions when prompted by the browser.

### 4. Start Detection

* Click **Start Webcam**
* Click **Start Detection**
* View real-time object detection results
